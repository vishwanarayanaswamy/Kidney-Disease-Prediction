This project focuses on predicting kidney disease using machine learning techniques. It includes a cleaned and balanced 
medical dataset, where missing values and inconsistencies have been handled, and the target variable has been adjusted to 
reduce bias. The repository contains multiple Jupyter notebooks (RQ1 to RQ7), each addressing a specific research question 
such as baseline model performance, model comparison, preprocessing effects, feature importance, hyperparameter tuning, 
cross-validation, and final model selection. In addition to the notebooks, the project also includes an outputs folder with 
generated tables (CSV) and figures (PDF) for each analysis. To run the project, make sure the dataset file is placed in the 
same directory as the notebooks, open any notebook in Jupyter, and execute all cells in order to reproduce the analysis and 
results.
