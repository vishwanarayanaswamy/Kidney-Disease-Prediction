This project focuses on predicting kidney disease using machine learning techniques. It includes a cleaned and balanced medical dataset where inconsistencies were removed and the target variable was adjusted to reduce bias. The repository contains multiple Jupyter notebooks (RQ1 to RQ7), each addressing different research questions such as model performance, preprocessing, feature importance, and evaluation. An outputs folder is also included, which contains the generated tables (CSV) and figures (PDF) for each analysis. The complete project, including the dataset, notebooks, and outputs, is available as a ZIP file for easy download. To run the project, simply extract the ZIP file and open the notebooks in Jupyter to execute the cells.
