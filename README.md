This project focuses on predicting kidney disease using machine learning techniques. It includes a cleaned and balanced medical dataset where inconsistencies were removed and the target variable was adjusted to reduce bias. The repository contains multiple Jupyter notebooks (RQ1 to RQ7), each addressing different research questions such as model performance, preprocessing, feature importance, and evaluation. An outputs folder is also included, which contains the generated tables (CSV) and figures (PDF) for each analysis. Additionally, the complete project — including the dataset, notebooks, and outputs — is provided as a ZIP file for easy access and download. To run the project, download and extract the ZIP file, ensure the dataset is in the same directory as the notebooks, open any notebook in Jupyter, and execute all cells step by step to reproduce the analysis.
